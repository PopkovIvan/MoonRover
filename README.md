#MoonRover: выполнение лунной миссии

__Вся программа состоит из трех пакетов:__
* __com.dooglys.game.entities__ содержит классы с реализацией сущностей
* __com.dooglys.game.interfaces__ содержит интерфейсы сущностей
* __com.dooglys.game__ содержит движок программы

Была предпринята попытка разработать легко-расширяемую игру отделив реализацию от интерфейса. Таким образом, получилось, что есть главенствующий родитель "Entity", который содержит методы, присущие всем остальным сущностям, а именно:
* alive() - проверяет жива ли сущность
* nextStep() - метод, который вызывается после каждого хода сущности Actor(например, при каждом втором ходе Area бы сужалась или Block меняли координаты)
* геттеры - чтобы у каждой сущности были координаты

И есть наследники, у каждого из которых определены необходимые ему методы. __Actor__ - представляет объект, который двигается по карте. __Area__ - карта. __Block__ - препятствие. __MoonRover__ - реализация лунохода, __SimpleArea__ - реализация карты, __SimpleBlock__ - реализация препятствия.
__GameEngine__ - представляет собой ядро программы, оно управляет всеми объектами. 

Подход разделения интерфейса от реализации позволит, в будущем, создавать новые объекты с уникальным поведением не меняя архитектуры программы.
